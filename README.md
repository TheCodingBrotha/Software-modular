# Software Modular

Sistema de gestão ERP desktop desenvolvido em C# com interface WinForms e integração com banco de dados SQL.

## Visão geral

O **Software Modular** é um sistema ERP desktop pensado para centralizar rotinas administrativas e operacionais de diferentes tipos de negócio, como academias, mercados e consultorias. O projeto foi desenvolvido com foco em organização, escalabilidade e facilidade de manutenção, oferecendo uma base sólida para evolução futura.

A aplicação reúne funcionalidades essenciais para controle de clientes, pedidos, serviços e histórico de movimentações, com autenticação de acesso e estrutura preparada para expansão por módulos.

## Objetivo

O objetivo central do projeto é oferecer uma solução única e adaptável para diferentes modelos de negócio, permitindo:

* cadastro e gerenciamento de clientes;
* controle de pedidos e serviços;
* consulta e manutenção de registros;
* persistência segura das informações em banco de dados;
* organização de processos em uma interface desktop intuitiva.

## Tecnologias utilizadas

* C#
* .NET WinForms
* SQL
* Visual Studio 2022

## Funcionalidades

### Autenticação

* Tela de login para controle de acesso ao sistema.
* Estrutura preparada para validação de usuários.

### Cadastro e manutenção de dados

* CRUD completo para clientes, pedidos e serviços.
* Fluxos de criação, consulta, atualização e exclusão de registros.

### Persistência de dados

* Integração com banco de dados SQL.
* Armazenamento e manipulação de informações de forma organizada.

### Estrutura modular

* Organização pensada para adaptação a diferentes cenários de uso.
* Base flexível para novos módulos e regras de negócio.

## Arquitetura do projeto

O sistema foi projetado com uma arquitetura modular, facilitando a separação de responsabilidades entre interface, regras de negócio e acesso a dados. Essa abordagem contribui para:

* melhor manutenção do código;
* maior reutilização de componentes;
* inclusão de novas funcionalidades com menor impacto;
* adaptação do sistema a diferentes segmentos de mercado.

## Telas do sistema

O projeto conta com telas voltadas para:

* login de usuário;
* página inicial / menu principal;
* cadastro de clientes;
* gerenciamento de pedidos;
* gerenciamento de serviços;
* histórico de registros.
